# CardGame
A simple Java GUI card game

## Description
This is a Java GUI card game in which you play against the dealer.
You are initially given $100 dollars to start with and make bets every round. 
You are given three cards from the top of the shuffled deck and dealers draws another three from the top of the deck.
You can choose to replace each cards once.
A person with more special cards or sum of the cards divided by 10 and take the remainder and higher score leads to winning the game.
